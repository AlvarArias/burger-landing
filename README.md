# Burger Landing Page

A modern, responsive landing page for a burger shop. This project showcases premium burgers with a clean design, smooth animations, and a mobile-friendly layout.

## Features

- **Modern UI**: Sleek design with a curated color palette and typography.
- **Responsive Design**: Works perfectly across mobile, tablet, and desktop devices.
- **Menu Section**: Highlighted burger categories (Classic, Spicy, etc.).
- **Built with**: HTML5 and Vanilla CSS.

## Getting Started

To view the project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/AlvarArias/burger-landing.git
   ```
2. Open `index.html` in your favorite web browser.

## Project Structure

- `index.html`: Main structure of the landing page.
- `style.css`: Custom styles and responsive layout.
- `asset/`: Directory containing burger images (classic, hero, spicy).
